# Media Cleaner: быстрая шпаргалка

Безопасно сжимает фото и видео, ищет дубликаты. Оригиналы автоматически не удаляет.

## 1. Установка на Mac

Сначала проверьте, установлен ли Python:

```bash
python3 --version
```

Если Terminal показывает версию Python 3, ничего устанавливать не нужно. Если команда не найдена, скачайте актуальный Python 3 с официального сайта [python.org](https://www.python.org/downloads/macos/), откройте установочный файл и пройдите обычную установку. После этого закройте и снова откройте Terminal.

Откройте Terminal и выполните:

```bash
brew install ffmpeg
python3 -m pip install Pillow
```

## Как быстро перейти в папку

Перейти нужно **в папку программы** — ту, внутри которой лежит файл `media_cleaner.py`.

Например, если вы скачали папку `MediaCleanerProject`, откройте её в Finder и убедитесь, что внутри виден файл `media_cleaner.py`. Именно папку `MediaCleanerProject` нужно перетащить в Terminal. Папку с фотографиями на этом этапе перетаскивать не нужно: программа попросит её позже.

Самый простой способ:

1. Напишите `cd` и поставьте пробел:

```bash
cd 
```

2. Перетащите папку с программой (`MediaCleanerProject`) из Finder в окно Terminal.
3. Путь вставится автоматически. Нажмите Enter.

Получится примерно так:

```bash
cd /Users/имя/Documents/MediaCleanerProject
```

После перехода запустите программу:

```bash
python3 media_cleaner.py --wizard
```

Так же можно вставлять пути к видео, input- и output-папкам: дождитесь вопроса программы, перетащите файл или папку из Finder в Terminal и нажмите Enter. Пробелы в пути Terminal обработает автоматически.

## 2. Самый простой запуск

Перейдите в папку со скриптом:

```bash
cd "/путь/к/MediaCleanerProject"
```

Запустите пошаговый мастер:

```bash
python3 media_cleaner.py --wizard
```

Дальше выбирайте действие по номеру. Файл или папку можно перетащить из Finder прямо в Terminal.

## 3. Полезные команды

Проверить папку без изменений:

```bash
python3 media_cleaner.py "/путь/input" "/путь/output" --dry-run
```

Сжать папку:

```bash
python3 media_cleaner.py "/путь/input" "/путь/output"
```

Быстро сжать одно видео рядом с оригиналом:

```bash
python3 media_cleaner.py --quick-file "/путь/video.mov"
```

Только найти точные дубликаты:

```bash
python3 media_cleaner.py "/путь/input" "/путь/output" --duplicates-only
```

## Важно

- Для первого запуска используйте `--wizard`.
- Перед большой обработкой сначала запускайте `--dry-run`.
- Input и output должны быть разными папками.
- Для каждого нового запуска лучше выбирать новую output-папку.
- Программа не удаляет файлы навсегда.
